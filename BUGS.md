## 2026-05-19 — 曲线笔刷转弯处断裂
- **症状**：曲线模式画弧线、转弯，外侧出现明显空白缝；笔触越粗、转弯越急越明显
- **根因**：`drawCurvedStretch` 每段中行用**自段切线**做旋转，相邻段在公共端点旋转方向不同 → 外侧缝隙 ≈ `(Wd/2)·tan(Δθ/2)`，笔触 200px、转弯 30° 时缝隙达 27px。原 `+1px overlap` 完全不够
- **修法**：每个内部 path 点加一片**平分角方向**中行 stamp（圆角接头），高度按 `Wd·tan(|Δθ|/2)` 自适应；同时把采样从 2.5px 缩到 1.5px 减小单段角度差
- **文件**：[index.html](index.html) line 376（接缝补丁）/ line 445（采样阈值）
